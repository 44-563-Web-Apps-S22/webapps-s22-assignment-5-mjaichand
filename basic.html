I have opened the html page in browser and started reducing the size of browser slowly. I observed that the the bird names breaks first. the list of all birds using col-md class and letters using col-sm-2 class. col-md class breaks if Screen width is less than 768px while the col-sm-2 class breaks if the screen width is less than 576px, which is less than that of col-md class. Hence the bird names breaks first.
